Orthogonal Matching Pursuit 2D in Java.

Orthogonal Matching Pursuit in 2D (OMP2D) is an effective algorithm for sparse image representation. It is dedicated to approximating an image as a superposition of elements chosen from two large sets, one for each dimension, celled dictionaries.

Installing the plugin for ImageJ

  1. First Download the ImageJ application from http://rsb.info.nih.gov/ij/download.html and the OMP2D plugin from here
  2. Extract ImageJ to a folder of your choice and copy OMP2D_Plugin.jar to the plugins folder within.
  3. Launch the ImageJ executable and you should find OMP2D listed under the plugins menu.

or if you're running 64-bit Linux

$ wget http://rsb.info.nih.gov/ij/download/linux/ij147-linux64.zip
$ unzip ij147-linux64.zip
$ wget -P ImageJ/plugins https://github.com/simonsm1/OMP2D/raw/master/OMP2D/OMP2D_Plugin.jar
$ ./ImageJ/ImageJ
